# 6502-OS
Lightweight OS for breadboard computer from the Ben Eater YT series.


I am trying to learn about OS-design, and wanted to create a cool custom OS for my breadboard computer. For now, it'll just be two threads alternating and printing to a screen- but in future I hope to make it far more complex...
